## 🎯 대표 사례

### Netflix: DVD → Streaming → Content
**전략 진화**
- 1997: DVD 우편 배송
- 2007: 스트리밍 전환
- 2013: 오리지널 콘텐츠

**성공 요인**
- 기술 변화 선제 대응
- 기존 사업 카니발라이제이션 감수
- 데이터 기반 콘텐츠 전략
- 글로벌 확장

### Spotify: Freemium Master
**전략 선택**
- 무료 + 프리미엄 모델
- 음악 산업과의 파트너십
- 플레이리스트 중심 경험
- 팟캐스트 확장

**교훈**
- Freemium의 전환율 관리
- 콘텐츠 비용 구조
- 네트워크 효과 창출
- 인접 영역 확장

### Blockbuster: 변화 거부의 대가
**실패 과정**
- Netflix 인수 기회 거절 (2000년)
- 온라인 전환 지연
- 연체료 비즈니스 모델 고집
- 2010년 파산

**실패 원인**
- Innovator's Dilemma
- 기존 수익 모델 집착
- 고객 행동 변화 무시
- 기술 변화 과소평가

## 🚀 Pivot 사례

### Successful Pivots

**Instagram: Burbn → Photo Sharing**
- 원래: 위치 기반 체크인 앱
- 발견: 사진 공유 기능만 인기
- 피봇: 사진 전문 앱
- 결과: Facebook이 $1B 인수

**Slack: Game → Communication**
- 원래: 게임 회사 내부 도구
- 발견: 커뮤니케이션 도구 가치
- 피봇: B2B SaaS
- 결과: $27B 가치 평가

**Twitter: Odeo → Microblogging**
- 원래: 팟캐스트 플랫폼
- 위기: iTunes 팟캐스트 출시
- 피봇: 140자 메시지
- 결과: 글로벌 플랫폼

### Failed Pivots

**Quibi: Premium Short-form**
- 가정: 통근 시간 시청
- 현실: COVID로 재택
- 문제: 유료 + 짧은 콘텐츠
- 결과: 6개월 만에 종료

## 🎓 전략적 교훈

### 타이밍의 중요성
- Too Early: Google Glass, Segway
- Perfect: Uber, Airbnb (2008 금융위기)
- Too Late: Google+, Microsoft Mobile

### 경쟁 우위 구축
- **Network Effects**: Facebook, LinkedIn
- **Switching Costs**: Adobe, Salesforce
- **Brand**: Apple, Tesla
- **Scale**: Amazon, Walmart

### 실행력의 차이
- 전략 10% + 실행 90%
- Fast iteration beats perfect planning
- Customer feedback > Internal opinion
