## 💭 자주 묻는 질문

**Q: PM이 되려면 기술 배경이 필요한가요?**
- 도움은 되지만 필수는 아닙니다
- 오히려 다양한 배경이 강점이 될 수 있습니다
- 중요한 것은 학습 능력과 커뮤니케이션

**Q: 주니어 PM은 뭘 하나요?**
- 회사마다 다르지만 보통 기능 단위 담당
- 시니어 PM의 멘토링 아래 성장
- 데이터 분석, 문서 작성, 실행 중심

**Q: PM과 PO의 차이는?**
- 조직마다 다른 정의
- 일반적으로 PM이 더 넓은 범위
- 많은 곳에서 구분 없이 사용
