## 핵심 역량

1. **데이터 리터러시**: 데이터를 읽고 해석하는 능력
2. **분석 도구 활용**: SQL, 엑셀, 분석 툴 활용
3. **인사이트 도출**: 데이터에서 의미있는 패턴 발견
4. **시각화**: 데이터를 효과적으로 전달
